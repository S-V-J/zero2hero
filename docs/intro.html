Getting Started

Ready to begin your journey from zero to hero? Let's set up your professional development environment.

✅ Prerequisites

  • ✓ Windows 10/11 (for WSL2) or macOS/Linux operating system
  • ✓ Basic computer literacy and file management skills
  • ✓ Willingness to learn and build real projects
  • ✓ Enthusiasm for open-source development
  • ✓ ~20GB free disk space for development tools

📋 Installation Checklist 

# 1. Install WSL2 (Windows)
wsl --install

# 2. Fork & Clone the Repository
git clone git@github.com:YOUR_USERNAME/zero2hero.git
cd zero2hero

# 3. Open in VS Code
code .

# 4. Verify Docker Installation
docker run hello-world

# 5. Join Discord for Setup Help
