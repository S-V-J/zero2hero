## Why This Course Exists

Most programming tutorials teach syntax in isolation. You learn Python, forget C. You build a todo app, then panic when asked to deploy a secure, scalable system.

**This course fixes that.**

Every concept is taught:
- Across multiple languages (C, C++, Python, JavaScript)
- Integrated into real production projects
- Hardened with cybersecurity from day one
- Deployed with professional DevOps workflows

This is not a collection of random tutorials. This is a **cohesive engineering curriculum** designed to transform beginners into production-ready developers.

## Course Roadmap

### Phase 1: Environment & Foundations
- WSL2 Ubuntu + VS Code setup
- Git + SSH + monorepo workflow
- Linux CLI mastery + Bash automation
- Docker + Docker Compose basics

### Phase 2: Core Programming & Algorithms
- C: pointers, memory management, Makefiles
- C++: OOP, STL, modern C++ (17/20)
- Python: venv, packaging, async, type hints
- Algorithms taught across all three languages

### Phase 3: Full Stack Development
- Frontend: React/Next.js + TypeScript
- Backend: FastAPI/Node.js + PostgreSQL
- APIs: REST, GraphQL, WebSockets
- DevOps: Docker, CI/CD, Nginx, deployment

### Phase 4: AI + ML Integration
- Python data stack: numpy, pandas, scikit-learn
- Deep learning: PyTorch/TensorFlow
- LLM integration: LangChain, Ollama, RAG
- Deploying ML models with FastAPI + Docker

### Phase 5: Enterprise VoIP (Asterisk + Kamailio)
- SIP/RTP fundamentals + Wireshark lab
- Kamailio: SIP proxy, routing, load balancing
- Asterisk: dialplans, IVR, AGI/ARI
- WebRTC + SIP over WSS integration

### Phase 6: Full-Stack Cybersecurity
- OWASP Top 10 mitigation + secure headers
- Auth: JWT, OAuth2, RBAC, MFA
- VoIP security: SRTP, TLS, anti-fraud
- Security testing: SAST/DAST, pentest basics
